📋 Installation Parameters

💡 Microinverter Home Run Design

For microinverter installations:

  • Multiple Microinverters: Select this when sizing the home run from a junction box that combines multiple microinverters
  • Typical Setup: 4-10 microinverters → Junction Box → Home Run to Main Panel
  • Current Calculation: Sum of all microinverter AC output currents
  • NEC Compliance: Each microinverter output is typically 1.2-1.6A at 240V AC
  • Wire Type: Use THWN-2 copper for AC home runs

💡 What This Tool Determines:

This calculator finds the minimum wire size based on:

  • Current carrying capacity (ampacity) per NEC Table 310.16
  • Voltage drop limitations (typically 3% max)
  • Temperature derating factors (NEC Table 310.15(B)(1))
  • Bundling/conduit fill adjustments (NEC Table 310.15(C)(1))
  • Solar-specific requirements (NEC Article 690)
  • Rooftop installation temperature adders
